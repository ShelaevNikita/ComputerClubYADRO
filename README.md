
# Моделирование работы компьютерного клуба

## Сборка программы
Программа разработана на ЯП C++ (ISO C++ 17).   
Чтобы скомпилировать исходные файлы, можно воспользоваться следующей командой:   
`g++ -o test.exe ComputerClubStart.cpp include/ComputerClub.h src/ComputerClub.cpp`,   
где `test.exe` - название исполняемого файла.   

## Использование программы
При запуске программы требуется передать ей в качестве аргумента командной строки название входного файла.
Сделать это можно подобным образом (где `examples/testFile1.txt` - название требуемого файла):
`./test.exe examples/testFile1.txt` - для вывода результата в консоль или
`./test.exe examples/testFile1.txt > output.txt` - для вывода результата в определённый файл (например, `output.txt`).

## События компьютерного клуба
### Входящие события
+ ID1 - Клиент пришёл: *<Время> 1 <Имя клиента>*
+ ID2 - Клиент сел за стол: *<Время> 2 <Имя клиента> <Номер стола>*
+ ID3 - Клиент ожидает: *<Время> 3 <Имя клиента>*
+ ID4 - Клиент ушёл: *<Время> 4 <Имя клиента>*
### Исходящие события
+ ID11 - Клиент ушёл: *<Время> 11 <Имя клиента>*
+ ID12 - Клиент сел за стол: *<Время> 12 <Имя клиента> <Номер стола>*
+ ID13 - Ошибка: *<Время> 13 <Ошибка>*
### Ошибки
+ **YouShallNotPass** - Клиент уже находится в компьютерном клубе (ID1)
+ **NotOpenYet** - Клиент пришёл в нерабочие часы (ID1)
+ **PlaceIsBusy** - Клиент пытается пересесть за занятый стол (ID2)
+ **ClientUnknown** - Клиент не находится в компьютерном клубе (ID2, ID4)
+ **ICanWaitNoLonger!** - В клубе есть свободные столы (ID3)
